# Resume Export Guide — ATS-Friendly DOCX/PDF

Use [Resume-ATS-Optimized.txt](Resume-ATS-Optimized.txt) as the source. Target ATS score after export: **78–82 / 100**.

---

## Before you export

1. Replace `[your-handle]` in the contact line with your real LinkedIn and GitHub URLs.
2. Confirm the career note (Oct 2020 – Feb 2022) matches your timeline; edit if you had contract/freelance work.
3. Add metrics where you have them (e.g., defect reduction %, release count per quarter).
4. Remove any bullet you cannot speak to in an interview.

---

## Word / Google Docs setup (recommended for job portals)

| Setting | Value |
|---------|--------|
| Layout | **Single column** only |
| Font | Arial or Calibri, **11 pt** body; **12 pt** name |
| Margins | 0.5–0.75 inch all sides |
| Line spacing | Single or 1.15 |
| Bullets | Standard round bullets (not custom icons) |
| Headers | ALL CAPS optional: `PROFESSIONAL SUMMARY`, `TECHNICAL SKILLS`, etc. |
| Length | 2 pages max for 4+ YOE with this content density |

### Section order (do not change)

1. Contact block (one line or stacked — no side-by-side columns)
2. Professional Summary
3. Technical Skills
4. Professional Experience (reverse chronological)
5. Career Note (if kept)
6. Education
7. Awards

### Do not use

- Tables for layout or skills
- Text boxes, shapes, or icons
- Skill bars or star ratings
- Multi-column templates (Canva, Novoresume two-column, etc.)
- Header/footer for contact info only (ATS may skip footers)

## File formats by use case

| Use case | Format | Notes |
|----------|--------|--------|
| Company ATS / Naukri / LinkedIn Easy Apply | **.docx** | Best parse rate; upload this version |
| Email to recruiter | **PDF** | Export from Word “Save as PDF”; ensure text is selectable |
| Quick paste | **.txt** | Use `Resume-ATS-Optimized.txt` directly |

### PDF export checklist

- [ ] Text is selectable (not a scanned image)
- [ ] No fused words (check “Constructed scalable” not “Constructedscalable”)
- [ ] Contact email and phone copy correctly when you highlight all text
- [ ] File size under 2 MB

## Quick re-score checklist

After export, open the PDF in a viewer, select all text, and verify:

- [ ] Name, email, phone appear once at the top
- [ ] Job titles and dates parse as plain text (e.g., `02/2022 – Present`)
- [ ] Skills are not mixed with degree names
- [ ] Metamorphosys appears before Tata Digital
- [ ] No sports or unrelated accomplishments in tech uploads
